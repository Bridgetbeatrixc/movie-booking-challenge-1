# Administration Frontend Feature

Owner: `Isi nama anggota 5`

Use this folder for admin dashboard, manage movies, manage showtimes, and all-bookings monitoring.

## Suggested Files

- `pages/` - Dashboard, manage movies, manage showtimes, and all bookings.
- `components/` - Admin layout, sidebar, stat cards, forms, and tables.
- `services/` - Admin API calls.

## Integration

- Uses role guard from `features/auth`.
- Reuses movie, showtime, and booking API contracts.
