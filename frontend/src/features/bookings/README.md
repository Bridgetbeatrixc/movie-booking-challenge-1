# Booking Frontend Feature

Owner: `Isi nama anggota 4`

Use this folder for final booking confirmation, user booking history, booking detail, cancellation, and conflict feedback.

## Suggested Files

- `pages/` - My bookings, booking detail, and confirmation pages.
- `components/` - Booking card, cancel dialog, status badge, and conflict feedback.
- `services/` - Booking API calls.

## Integration

- Receives selected showtime and seats from `features/showtimes`.
- Uses authenticated user state from `features/auth`.
