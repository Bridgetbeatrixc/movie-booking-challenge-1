# Beatrix Movie

Clean two-folder project structure:

- `frontend/` — React + Vite movie booking UI
- `backend/` — Express + Mongoose MongoDB API template

## Run frontend

```bash
cd frontend
npm install
npm run dev
```

## Run backend

```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

Frontend default: `http://localhost:5173`  
Backend default: `http://localhost:5000`
