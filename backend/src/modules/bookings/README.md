# Booking Backend Module

Owner: `Isi nama anggota 4`

This module owns booking creation, booking history, booking detail, cancellation, and conflict handling.

## Current Files

- `booking.model.js` - current Mongoose schema for booking data.
- `booking.controller.js` - current list and create logic.
- `booking.routes.js` - current `/api/bookings` routes.

## Planned Endpoints

- `POST /api/bookings`
- `GET /api/bookings/me`
- `GET /api/bookings/:id`
- `DELETE /api/bookings/:id`
- `GET /api/admin/bookings`
