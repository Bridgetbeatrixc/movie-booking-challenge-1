# Movie Backend Module

Owner: `Isi nama anggota 2`

This module owns movie catalogue data and movie API endpoints.

## Files

- `movie.model.js` - Mongoose schema for movies.
- `movie.controller.js` - movie list and create logic.
- `movie.routes.js` - `/api/movies` routes and nested showtime route.

## Endpoints

- `GET /api/movies`
- `POST /api/movies`
- `GET /api/movies/:movieId/showtimes`
